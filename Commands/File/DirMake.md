# DirMake

Creates a directory.

## Syntax

```pebakery
DirMake,<DestDir>
```

### Arguments

| Argument | Description |
| --- | --- |
| DestDir | Full path of the directory to create. |

## Remarks

## Examples

### Example 1

```pebakery
// C:\Temp will be created.
DirMake,C:\Temp
```
